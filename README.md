# 📊 FNP Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes sales data for Ferns and Petals (FNP), focusing on revenue trends, customer behavior, product performance, and delivery efficiency.

The dashboard is built using **Microsoft Excel (Power Pivot, Pivot Tables, and Data Visualization)**.

## 📊 Key Insights

- Anniversary and Raksha Bandhan generate highest revenue  
- Colors and Soft Toys are top-performing categories  
- Tier-2 cities contribute major share of orders  
- Sales peak during festive months like February and August  
- Slight increase in delivery time with higher order quantities
